# PyASCIIArt

[English](README.md)

PyASCIIArt 是一个用于创建 ASCII 艺术和动画的 Python 库。

## 模块

-   `ascii_renderer`: 这是项目的主要模块，负责渲染 ASCII 艺术。

## 示例
-   `demo_animation`: 一个展示动画功能的示例项目。
-   `flappy_bird`: 一个使用 PyASCIIArt 构建的示例游戏项目。
